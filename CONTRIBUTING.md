# Contributing to CtxPack

First off, thank you for considering contributing to CtxPack! It's people like you that make CtxPack such a great tool for the AI community.

## Where to start?

If you're new to the project, a great way to start is by:
- Trying out the demos and reporting any issues.
- Improving the documentation.
- Adding support for new OCI registries.

## How can I contribute?

### Reporting Bugs
- Use the GitHub Issue tracker.
- Describe the bug and include steps to reproduce it.

### Suggesting Enhancements
- Open a new Issue with the tag `enhancement`.
- Explain why the enhancement would be useful.

### Pull Requests
1. Fork the repo.
2. Create a new branch (`git checkout -b feature/amazing-feature`).
3. Make your changes.
4. Commit your changes (`git commit -m 'Add amazing feature'`).
5. Push to the branch (`git push origin feature/amazing-feature`).
6. Open a Pull Request.

## Tech Stack
- Python 3.8+
- `requests` library
- OCI / Docker Registry API V2

## Roadmap Priorities
- **Cache GC:** Implementing TTL and size-based purging.
- **Parallel Hashing:** Improving performance for large datasets.
- **Multi-layer Support:** Allowing delta updates to packs.
